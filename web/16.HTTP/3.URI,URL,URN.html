1.URL(Uniform Resource Identifier/统一资源标志符)

  • 用来唯一标识互联网上的信息资源
  • 包括URL 和 URN, URL 和 URN 都是 URI 的子集 ,URL和URN都是URI，但是URI不一定是URL或者URN。

2.（Uniform Resource Locator/统一资源标志符）URL是URI的一种，不仅标识了Web 资源， 还指定了操作或者获取方式，
同时指出了主要访问机制和网络位置。
其中的 ftp://ftp.is.co.za/rfc/rfc1808.txt 和 http://www.cnblogs.com/nods/p/8985322.html 都是 URL。
http:// 是协议 ( 文章最下方给出了常用的几种模式或协议 )。
www.cnblogs.com/nods/p/8985322.html 是网络资源的具体位置 URL就是 告诉你资源文件在网络上的具体位置。

3.URN Uniform Resource Name（统一资源命名）其中的 ftp.is.co.za/rfc/rfc1808.txt 和 www.cnblogs.com/nods/p/8985322.html#position 是URN。

  • http:// 是协议 ( 文章最下方给出了常用的几种模式或协议 )。
  • www.cnblogs.com/nods/p/8985322.html 是网络资源的具体位置。
  • #position 则是资源。

URL类似于住址，它告诉你一种寻找目标的方式(在这个例子中，是通过街道地址找到一个人)。要知道，上述定义同时也是一个URI。 URN = Universal Resource Name 统一资源名称 我们可以把一个人的名字看作是URN;因此可以用URN来唯一标识一个实体
